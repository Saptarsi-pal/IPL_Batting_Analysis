# IPL_Batting_Analysis
Data Analysis Project Showcasing IPL 2025 Batting Performance Using Python, SQL, and Power BI


Overview

This project presents a complete data analytics workflow using IPL 2025 batting statistics. The aim was to transform raw cricket data into meaningful insights through data cleaning, exploratory analysis, SQL querying, dashboard development, and reporting.

The project demonstrates practical skills in Python, MySQL, Power BI, and business storytelling, making it a strong portfolio project for data analyst roles.

Dataset

The dataset contains batting performance data of players from IPL 2025.

Key Attributes:
Player Name
Team Name
Matches Played
Innings
Runs Scored
Highest Score
Batting Average
Strike Rate
Balls Faced
Fours & Sixes
Half-Centuries
Centuries
Dataset Size:
Rows: 156
Columns: 14
Tools & Technologies
Python – Data processing and analysis
Pandas / NumPy – Data manipulation
Matplotlib / Seaborn – Data visualization
MySQL – Querying and analysis
Power BI – Interactive dashboard creation
Gamma – Presentation design
MS Excel / CSV – Data storage format
Project Workflow
1. Data Loading
Imported dataset using Python.
Verified structure, column names, and data types.
2. Exploratory Data Analysis (EDA)
Performed statistical summaries.
Identified trends, distributions, and performance patterns.
Checked for null values and inconsistencies.
3. Data Cleaning
Renamed columns for consistency.
Standardized formats.
Ensured clean and analysis-ready data.
4. Feature Engineering

Created calculated metrics such as:

Runs per Match
Boundary Runs
Boundary Percentage
5. SQL Analysis (MySQL)

Solved analytical business questions including:

Top 10 run scorers
Highest batting average
Best strike rate players
Most centuries / half-centuries
Team-wise total runs
Most sixes by team
Highest individual score
6. Dashboard Development

Designed an interactive Power BI dashboard for performance tracking and comparison.

7. Reporting & Presentation

Prepared final project documentation and presentation slides using Gamma.

Conclusion

This project showcases an end-to-end Data Analytics lifecycle — from raw data handling to dashboard reporting. It reflects strong practical knowledge in Python, SQL, Power BI, and analytical thinking, valuable for Data Analyst and Business Intelligence roles.
